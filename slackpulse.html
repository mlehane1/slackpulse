🔒 Privacy: This app only reads public & private channels using channels:history and groups:history scopes. Direct messages (DMs) are never accessed.
▸ ONE-TIME SETUP (~2 min)
Slack blocks direct browser requests. Run this local proxy:
$ mkdir slack-proxy && cd slack-proxy
$ npm init -y && npm install express cors node-fetch
# Place proxy.js here, then:
$ node proxy.js
✅ Proxy running at http://localhost:3001

· channels:read  · channels:history
· groups:read   · groups:history
· users:read
